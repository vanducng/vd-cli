# Architecture Decision Records

Lightweight ADRs capturing decisions that are costly to reverse. One decision per file, numbered, append-only (supersede rather than edit history).

Format: Context → Decision → Consequences → Alternatives rejected. Status is one of `Proposed | Accepted | Superseded by NNNN | Deprecated`.
